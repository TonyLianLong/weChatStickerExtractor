# weChatStickerExtractor

## 说明
此程序仅作为一个批量复制无加密文件工具，不含有任何解密新版微信加密表情包文件的功能，仅供学习用途，若需要使用此工具请使用2.0b4.0.9版微信。

Mac微信表情包提取工具，提取动态的表情包~ <br/>
